- в tools добавлен скрипт momo.sh для ручной обработки всех омографов, включает в себя зародыш скрипта для автоматической обработки
омографов, аналогичный 20-index.sed, но цель его - только уменьшить количнество обработки, не внося, по возможности, ложных

- в скриптах momo.sh, yofik.sh, get-words.sh реализован сходный функционал по ручной обработке омографов (соответственно: всех, ё-омографов,
имён-омографов). Генерируется набор скриптов, куда влючено: команды обработки на выбор, инфо в "стандартном" правиле из lexx, а также
все вхождения слова из обрабатываемой книги в контексте.
Всё это работает только при установленных плагинах для vim (vim-PatternsOnText и vim-ingo-library). Читайте комментарии в самих скриптах.
